>     肺结节（pulmonary nodule）的医学定义：边界清楚的、影像学不透明的、直径≤3cm、周围完全被含气肺组织包绕的单发或多发的肺部结节，不伴有肺不张、肺门肿大和胸腔积液。（当直径>3cm，就称为“肿块”了~，一般都是恶性的）
